Como transformar uma string em letras minúsculas?

Você pode usar o método lower() para converter todas as letras de uma string para minúsculas. Por exemplo: 

titulo = "Aprendendo Python"
titulo_minusculo = titulo.lower()

Como substituir espaços por hifens em uma string?

Você pode usar o método replace() para substituir espaços por hifens. Por exemplo: 

titulo_com_hifens = titulo.replace(" ", "-")

Posso usar os dois métodos em uma única linha?

Sim, é possível. Exemplo: url_amigavel = titulo.lower().replace(" ", "-")
