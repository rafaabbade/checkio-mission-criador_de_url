Enunciado

Marta, responsável pelo marketing digital de uma pequena livraria, quer transformar títulos de postagens de blog em URLs amigáveis. Crie uma função que converta um título de artigo em uma URL amigável. O título deve ser convertido para letras minúsculas, e os espaços devem ser substituídos por hifens.

Entrada: Uma string (titulo_artigo).

Saída: Uma string representando a URL.
